# Archive Index

This directory is cold storage for superseded, historical, or version-specific
material.

Use the archive when you need:

- design history
- older planning context
- retired reference notes
- past audits
- version-era working notes

Do not treat archive documents as current guidance unless an active document
explicitly points to them for history.

## Archive sections

- [`docs/archive/design/README.md`](design/README.md) — retired design notes and implementation-history briefs
- [`docs/archive/reference/README.md`](reference/README.md) — retired reference companions and deep-dive notes
- [`docs/archive/work/README.md`](work/README.md) — superseded planning and operational briefs
- [`docs/archive/versioned/README.md`](versioned/README.md) — version-era planning and worked-example material
- [`docs/archive/audits/README.md`](audits/README.md) — architecture/codebase audit reports
- [`docs/archive/issues/README.md`](issues/README.md) — issue-specific historical notes retained for context
