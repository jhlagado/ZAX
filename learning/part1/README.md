# Part 1 — The Computer

No prior knowledge of computers or programming assumed.

This part answers the questions that must be settled before any code makes sense: what is a computer, what is a program, and what does an assembler actually do? Three short chapters. By the end you will have a concrete mental picture of the Z80 and a clear reason for everything that follows.

Continue with [Part 2 — Learn Z80 Programming in ZAX](../part2/README.md) when you are done.
